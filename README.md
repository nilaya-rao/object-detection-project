\# Object Detection Web Application (Django + YOLOv8)



\## 📌 Project Overview

This project is a web-based object detection system built using Django and YOLOv8. It allows users to upload images and detect objects in real-time using a deep learning model.



\---



\## 🚀 Features

\- Upload images through web interface

\- Detect objects using YOLOv8 model

\- Display results with bounding boxes

\- User-friendly interface using Django templates



\---



\## 🛠️ Technologies Used

\- Python

\- Django

\- YOLOv8 (Ultralytics)

\- OpenCV

\- HTML, CSS

\## ⚙️ Installation \& Setup



1\. Clone the repository:

git clone https://github.com/nilaya-rao/object-detection-project.git



2\. Navigate to project:

cd object-detection-project



3\. Install dependencies:

pip install -r requirements.txt



4\. Run the server:

python manage.py runserver



5\. Open in browser:

http://127.0.0.1:8000/
